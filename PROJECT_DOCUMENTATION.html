🎯 Project Overview

TradeMind AI • AI-Powered Trading Platform

TradeMind AI is a comprehensive web-based trading platform that combines real-time market data streaming, machine learning price predictions, and AI-powered sentiment analysis to provide traders with intelligent insights for multiple asset classes.

🚀 Key Capabilities:
• Real-time market data from Binance WebSocket and simulated sources
• 6 ML models (LSTM, XGBoost, Random Forest, RNN, AdaBoost, ARIMA)
• Google Gemini AI sentiment analysis
• Live trading with Market/Limit orders
• Portfolio management with P&L tracking
• Support for 18+ tradeable symbols across 4 market categories

AI Sentiment Analysis Workflow:

User clicks "Analyze" → aiService.analyzeStockSentiment() retrieves API key → Calls Google Gemini API → Analyzes latest market news → Generates sentiment score (0-100) and market analysis → Results displayed in MLPanel → User reviews insights and makes trading decisions

Workflow Steps Breakdown

1

User Authentication

User logs in via LoginScreen → Credentials validated by authService → User stored in localStorage & AuthContext

2

Market Selection

User chooses market category (NIFTY 50, FOREX, CRYPTO, US MARKETS) and selects a symbol → App state updated

3

Data Subscription

streamService.subscribe() initiated → If Crypto: connects to Binance WebSocket → Otherwise: starts simulated data generation

4

Real-Time Data Updates

OHLCV data streamed via callbacks → TradingChart renders candles → dataService calculates ML metrics

5

ML Predictions

6 models process data → Generate Buy/Sell/Hold signals with confidence → Display in MLPanel

6

Order Placement

User creates order in OrderPanel → Market order executes immediately → Position added to portfolio

7

Position Management

Open positions tracked in real-time → User can close positions → P&L calculated and moved to trade history

8

Data Persistence

All portfolio changes automatically saved via AuthContext → Data persists across sessions in localStorage

Data Integration

🔌 Binance WebSocket (Cryptocurrency)

For cryptocurrency trading, the platform connects to Binance WebSocket API using the kline endpoint:

URL: wss://stream.binance.com:9443/ws/{pair}@kline_{interval} Example: wss://stream.binance.com:9443/ws/btcusdt@kline_1m Features: • Real-time price updates • Live OHLCV data • Multiple timeframe support (1m, 5m, 15m, 30m, 1h, 4h, 1d, 1w) • Automatic candle aggregation

📊 Simulated Data (Stocks, Forex, US Markets)

For instruments without direct API feeds, the platform uses realistic simulated data:

Generation Method: • Random walk model with trend • Volatility-aware price movements • Respects market hours (closed outside trading hours) • Generates realistic open/high/low/close values Update Interval: Configurable per timeframe • 5 seconds to 5 years supported • Historical data caching

Data Processing Pipeline

Raw Data → Validation → OHLCV Aggregation → ML Processing → Chart Rendering

1. Raw Data: Received from Binance WebSocket or generated via simulation
2. Validation: Check timestamp, price bounds, volume
3. Aggregation: Combine ticks into candles based on selected timeframe
4. ML Processing: 6 models generate predictions simultaneously
5. Rendering: Chart updates in real-time with animations
